Depression diagnosis remains susceptible to subjective bias, underscoring the urgent need for objective, biomarker-driven recognition methods.Electroencephalography (EEG) captures real-time cortical activity and holds promise as a non-invasive diagnostic adjunct.
In this study, we leverage the publicly available MODMA dataset and employ the phase lag index (PLI) to quantify pairwise electrode correlations, thereby constructing functional connectivity matrices for automated depression recognition.Our primary objective is to identify discriminative connectivity features that enhance classification accuracy.The dataset comprises 24 patients with major depressive disorder (MDD) and 29 healthy controls (HC), each contributing ten 10-second resting-state EEG segments, yielding 530 samples.Each sample is represented by 5,460 PLI-based functional connectivity features derived from all pairwise connections among 105 electrodes.We first establish two baseline classifiers---a support vector machine (SVM) with a radial basis function kernel and a multi-layer perceptron (MLP)---and subsequently develop an enhanced SVM model that incorporates ANOVA F-test-based feature selection and grid-search hyperparameter optimization.Six evaluation metrics are adopted: accuracy (ACC), sensitivity (SEN), specificity (SPE), precision (PRE), F1-score, and area under the ROC curve (AUC), all assessed via stratified ten-fold cross-validation.Experimental results demonstrate that the SVM baseline achieves an accuracy of 86.23% and an AUC of 93.36%, while the MLP, constrained by limited sample size, reaches an accuracy of 82.08%.The enhanced model outperforms the SVM baseline in sensitivity (82.50%) and AUC (93.94%), confirming that eliminating redundant connectivity features effectively improves depression recognition performance.
